# TD-CMAKit

**磨刀不误砍柴工**

2020/2021(2) 计算机组成原理 TD-CMA 指令微指令相关工具

目前实现

- 微指令的汇编器

## 例子

### 微指令汇编

```
.START:
AR=PC
IR=MEM
<P1:30>
	0:ADD
	2:IN
	4:OUT
	5:HLT
	C:JMP

.ADD:
	A=RS
	B=RD
	RD=A+B
	GOTO START

.IN:
	RD=IN
	GOTO START

.OUT:
	AR=MEM
	OUT=RS
	GOTO START

.HLT
	NOP
	GOTO HLT

.JMP:
	AR=PC
	PC=MEM
	GOTO START
```
